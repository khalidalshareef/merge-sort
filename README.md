# merge-sort

[16,21,11,8,12,22] -> Merge Sort

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.

Big-O gösterimini yazınız.

# Adımlar

[16, 21, 11, 8, 12, 22]

[16, 21, 11] and [8, 12, 22]

[16], [21, 11] and [8,12], [22]

[16], [21], [11] and [8], [12], [22]

[16] , [11, 21] and [8, 12] and [22]

[11, 16, 21] and [8, 12, 22]

[8, 11, 12, 16, 21, 22]

# Big O-Notation

O(nlogn)
